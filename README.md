# 🧠 Профиль компетенций
</div>
<details>
<summary><b>🌐 Browser DevTools + API & Protocols</b></summary>
<br>
Использую <b>Chrome DevTools</b> для отладки автотестов (анализ <i>DOM</i>, сетевых запросов) и ручного тестирования (адаптивность, уязвимости). Также использую <b>Postman</b> и <b>Swagger</b>.
Понимаю архитектуру <b>REST</b>, <b>GraphQL</b>, <b>WebSocket</b> и <b>gRPC</b>. 
Практикуюсь на публичных песочницах (например, <i>GraphQL Cards</i>), проверяя ответы сервера и структуру данных.
</details>

<details>
<summary><b>🔐 Security & Auth</b></summary>
<br>
Базовое тестирование безопасности: анализ <b>Security Headers</b>, проверка <b>CORS</b>. 
Знаю принципы моделирования <i>CSRF/XSS-атак</i> для проверки защищенности форм. Применял на тестовых страницах.
Работал с <b>Token Auth</b>, <b>Basic Auth</b>, проверял доступы к защищенным эндпоинтам на тестовых полигонах, как на отдельных демо страницах <b>Security Headers</b> и <b>CORS</b> - так и Postman коллекциях для <b>JWT</b>.
</details>

<details>
<summary><b>🤖 Automation & CI/CD</b></summary>
<br>
Основной фокус развития. Пишу автотесты на <b>Python</b> используя <b>Playwright</b> + <b>Pytest</b>. 
Изучаю и практикую структуризацию проектов по паттерну <i>Page Object</i>. 
Изучаю интеграцию тестов в пайплайны <b>GitLab CI</b> для автоматического запуска при билде.
</details>

<details>
<summary><b>💾 Databases</b></summary>
<br>
Работа с <b>PostgreSQL</b>. Умею формировать запросы <b>SQL</b> уровня <i>DML</i> и <b>JOIN</b> для валидации данных на бэкенде. 
Поддерживаю навык решением задач на тренажерах (например, <i>sql-academy</i>).
</details>

<details>
<summary><b>⚙️ Process & Quality</b></summary>
<br>
Применяю техники тест-дизайна (<i>Risk-Based</i>, <i>Exploratory</i>, <i>Session-Based</i>, <i>Equivalence Partitioning</i>). 
Грамотно оформляю артефакты: <b>Баг-репорты</b>, <b>Тест-кейсы</b>. 
Быстро адаптируюсь к новым процессам в команде и инструментам управления задачами.
</details>
